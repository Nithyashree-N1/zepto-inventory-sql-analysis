# SQL 

This directory contains the SQL code used for the Zepto Inventory Analysis project.

## File Description

* **zepto_inventory_analysis.sql** – Complete SQL workflow including:

  * Table creation
  * Data exploration
  * Data cleaning
  * Business analysis queries
  * Inventory and pricing insights

## Analysis Coverage

The SQL script answers the following business questions:

1. Products with the highest discount percentages.
2. Expensive products currently out of stock.
3. Estimated revenue by product category.
4. Premium products with low discounts.
5. Categories offering the highest average discounts.
6. Best-value products based on price per gram.
7. Product segmentation by weight category.
8. Total inventory weight by category.

The script was developed and executed using PostgreSQL.
